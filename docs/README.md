# Docs

| 문서 | 내용 |
|------|------|
| [schema/](./schema) | **DB 스키마 문서** (tbls 자동 생성) — 폴더를 열면 전체 ER 다이어그램과 테이블 목록이 바로 렌더링됩니다. 테이블별 페이지에서 해당 테이블 중심의 부분 ERD를 볼 수 있습니다. |

## DB 스키마 문서 갱신 방법

스키마(마이그레이션)를 변경하는 PR에서는 아래 스크립트를 실행해 `docs/schema/` 변경분을 함께 커밋하세요.

```bash
./scripts/generate-schema-docs.sh
```

깜빡하면 **Schema Docs** CI가 실패하면서 알려줍니다. (검증 모드 — CI는 문서를 자동 커밋하지 않습니다)
